리눅스 명령어를 연습중 입니다.

pwd(print working directory) : 현재 디렉터리
	- pwd
cd(change directory) : 디렉터리 변경
	- cd : 내 디렉터리로 이동
	- cd [dir_name] : 해당 디렉터리로 이동
ls(list)
	- ls -l : 파일 리스트 확인
	- ls -al : 숨김 파일을 포함한 전체 파일 리스트 확인
cp(copy)
	- cp [file_name] [new_file_name] : 파일 복사
mkdir(make directory)
	- mkdir [dir_name] : 디렉터리 생성
rmdir(remove directory)
	- rmdir [dir_name] : 빈 디렉터리 제거
touch
	- touch [file_name] : 파일 생성
rm(remove) : 제거
	- rm -rf [dir_name] : 비어있지 않은 디렉터리 제거
	- rm -f [file_name] : 파일 제거
mv(move) : 이동 및 변경
	- mv [file_name] [path] : 파일 이동
	- mv [file_name] [new_file_name] : 파일명 변경
vi
	- vi [file_name] : 파일 open
		- i(insert) : 입력
		- esc + :wq : 저장 및 종료
cat
		- cat [file_name] : 파일 내용 확인
